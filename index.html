Networking Project at American University

In this project, I divided the 172.16.10.0/24 network into eight subnets using a /27 subnet mask. The subnets were assigned to different office departments, and VLANs were configured for management, marketing, accounting, and native purposes. I set up trunking between switches, configured inter-VLAN routing on routers, and implemented port security on access ports.

Additionally, I enabled OSPF for dynamic routing across routers, created and applied access control lists (ACLs) to restrict unauthorized access, and configured network devices with secure authentication and encrypted passwords. Remote administration was enabled via SSHv2 to ensure secure access to network devices.
